# Sistem Sorting Manggis Berbasis Computer Vision dan Fuzzy Logic

Proyek ini merupakan tugas akhir semester 5 program studi Mekatronika dan Kecerdasan Buatan

Sistem ini dirancang untuk melakukan penyortiran buah manggis secara otomatis menggunakan:
- Algoritma Decision Tree untuk mengembangan model
- Computer Vision untuk ekstraksi fitur visual
- Fuzzy Logic (Metode Mamdani) untuk pengambilan keputusan pengolahan pasca panen
- Dashboard interaktif untuk monitoring hasil sorting

## Fitur Sistem
- Akuisisi citra manggis menggunakan kamera
- Preprocessing citra (filtering, segmentasi, ROI)
- Ekstraksi fitur visual (warna, area, dsb.)
- Penentuan kualitas manggis menggunakan **Fuzzy Mamdani**
- Dashboard visualisasi hasil sorting secara real-time
